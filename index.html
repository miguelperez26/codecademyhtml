Other HTML tags

There are plenty of tags we can use in HTML in order to display content differently. The most common tags are used for text. Titles, for example can have different tags depending on the title size, from <h1> which is the largest to <h6> which is the smallest.

Most Used HTML tags

Tag Name Description
H1Largest TitleThis is the most used HTML title tag as it shows the title in the largest size.
H2Second largest titleThis tag is used very often as well as it holds the second largest title size.
H3Third largest titleThis one is important too as it is the third largest title size.
pParagraphp is very important too as it holds a paragraph.
